03 / Details

Before you install.

Does Corewise require Full Disk Access?

No. Normal diagnostics work without it. Full Disk Access is optional and used only when you explicitly start read-only Full Storage Analysis.

Can Corewise see my cloud AI agents?

No. AI Workloads observes attributable local processes. It does not read prompts, projects, agent threads, cloud usage, process arguments, or working directories.

Does Corewise fix or delete anything?

No. Corewise explains supported signals and points to safe next steps. It does not kill processes, delete files, or claim a single health verdict.

How are updates delivered?

The first release line uses manual updates from GitHub Releases. An updater will not be added until its networking and trust model are explicitly reviewed.
